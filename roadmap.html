Data Science Roadmap

Complete Learning Path for ML / AI / Data Science

1 Mathematics

Linear Algebra

Scalars, vectors, matrices; Square matrices, identity matrix, triangular matrices; Unit vectors; Sparse vs dense matrices; Scalar multiplication; Vector and matrix addition; Matrix multiplication (rules and algorithms); Transpose, conjugate transpose; Symmetric, Hermitian, skew-Hermitian, unitary matrices; Rank, determinant, trace; Matrix inverse; Vector spaces; Basis and dimension; Span; Orthogonality and orthonormality; Linear transformations; Solving systems of linear equations; Gaussian elimination; Gauss–Jordan elimination; LU decomposition; QR decomposition; Eigenvalues and eigenvectors; Diagonalization; Singular Value Decomposition (SVD); Inner and outer products; Linear least squares

Calculus (Single & Multivariate)

Functions of a single variable; Functions of several variables; Limits; Continuity; Derivatives; Partial derivatives; Gradients; Jacobians; Hessians; Minimum and maximum of functions; Optimization intuition; Step function; Sigmoid function; Logit function; ReLU (Rectified Linear Unit); Softmax; Cost / loss functions

Probability Theory and Distributions

Probability axioms; Infinite sample spaces; Axiomatic probability; Measure theory; Random variables; Discrete and continuous distributions; Probability density functions; Expectation; Variance; Covariance; Conditional probability; Bayes' theorem; Conditional expectation; Least squares prediction; Law of Large Numbers; Central Limit Theorem; Poisson approximation; Poisson process; Brownian motion; Stochastic processes; Stationary processes; Markov chains; Markovian processes

Optimization Methods

Objective / cost functions; Error functions; Likelihood functions; Gradient Descent algorithm; Variants of Gradient Descent (batch, stochastic, mini-batch, etc.)

Advanced Optimization & Operations Research

Convex vs non-convex optimization; Linear & Integer Programming; Stochastic optimization; Constraint optimization; Reinforcement Learning + OR hybrids; Scheduling & resource allocation

2 Statistics

Descriptive Statistics

Measures of central tendency (mean, median, mode); Measures of dispersion (range, variance, standard deviation, IQR); Skewness; Kurtosis; Frequency tables; Summary statistics

Sampling & Estimation

Sampling techniques; Sampling distributions; Sample survey methods; Point estimation; Interval estimation; Confidence intervals

Inferential Statistics

Hypothesis testing; p-values; Significance testing; Parametric tests; Non-parametric tests; Analysis of Variance (ANOVA)

Regression & Correlation Analysis

Linear regression; Logistic regression; Multicollinearity; Model diagnostics; Correlation analysis

Time Series & Stochastic Data

Time series analysis ARIMA, SARIMA; State space models; Forecasting

Bayesian Statistics

Prior; Likelihood; Posterior; Bayesian inference; Bayesian methods; MCMC basics

Experimental Design

Experimental design principles; Factorial experiments; A/B testing; Power analysis; A/B testing at scale

Nonparametric & Robust Methods

Nonparametric methods

Statistical Process & Decision Analysis

Statistical process control; Decision analysis

Advanced Statistics & Causal Inference

Causal graphs (Directed Acyclic Graphs, DAGs); Do-calculus (conceptual); Instrumental variables; Propensity score matching; Difference-in-differences; Synthetic analysis; Regression discontinuity; Uplift modeling

Data Structures & Algorithms

Working with Libraries, Modules & Frameworks

Working with external libraries; Modules and frameworks; Scientific and data analysis libraries

Databases & Data Storage

Scientific Computing & Modeling

Development and modeling in MATLAB / Octave; Simulation and visualization of models; Brain circuits and neural networks modeling; Data-driven decision making; Pattern discovery and predictive modeling

Parallel & High-Performance Computing

Parallel computing; Performance optimization

Software Engineering Practices

Clean code principles; Tidy / readable code; Code optimization; Unit testing; Integration testing; Version control systems (Git); Command-line interface (CLI) usage; Integrated Development Environments (PyCharm, Anaconda, Jupyter Notebooks, etc.)

Model Deployment & Production

Model deployment; Packaging applications; Dependency management in production

Software & Web Development Best Practices

Software development best practices; Web development best practices

Web scraping; APIs; Database extraction (SQL / NoSQL); Scripting for data extraction from websites, APIs, and databases

5 Machine Learning

Foundations of Machine Learning

Learning paradigms: supervised, unsupervised, reinforcement; Objective / loss functions; Gradient Descent: batch, stochastic, mini-batch; Overfitting & underfitting; Bias–variance trade-off; Curse of dimensionality; No Free Lunch Theorem; Occam's Razor

Supervised Learning - Regression

Linear regression; Logistic regression; Ordinary Least Squares (OLS); Partial Least Squares (PLS); Gradient Boosting Regression; Multivariate Adaptive Regression Splines (MARS); ARIMA (time-series regression context)

Supervised Learning - Classification

k-Nearest Neighbors (k-NN); Naive Bayes; Support Vector Machines (SVM); Kernel methods; Radial Basis Functions (RBFs)

Statistical & Tree-Based Machine Learning

Decision Trees; Classification and Regression Trees (CART); ID3; C4.5, C5.0; CHAID; Learning Vector Quantization (LVQ)

Ensemble Methods

Bagging; Boosting; AdaBoost; Gradient Boosting (XGBoost, LightGBM); Random Forests; Voting classifiers; Stacking; Blending; Dropout (as ensemble regularization idea)

Unsupervised Learning - Clustering

K-means; Hierarchical clustering; DBSCAN

Dimensionality Reduction

Principal Component Analysis (PCA); Linear Discriminant Analysis (LDA); Kernel PCA (KPCA); t-SNE; UMAP

Model Evaluation & Diagnostics

Classification metrics: Accuracy; Confusion matrix; Precision; Recall; Specificity; F1-score; ROC-AUC; Regression metrics: RMSE, MAE, MAPE; Information criteria: AIC; BIC; Model diagnostics: Multicollinearity detection - Correlation matrix, Tolerance, Variance Inflation Factor (VIF)

Model Validation & Hyperparameter Tuning

Holdout validation; k-fold cross-validation; Resampling methods; Bootstrapping; Learning curves; Validation curves; Regularization: Ridge (L2); LASSO (L1); Hyperparameter search: Grid search; Randomized search

Reinforcement Learning

Markov Decision Processes (MDPs); Reward optimization; Q-learning; Monte Carlo methods; Policy gradients; Deep Q-Networks (DQN); TF-Agents
